# AI Daily Report 🤖

每天自动收集AI方向最新新闻并生成日报网站。

## 功能特性

- ✅ 每天8点自动收集AI/科技新闻
- ✅ 生成美观的日报网页
- ✅ 自动发布到GitHub Pages
- ✅ 历史日报可追溯
- ✅ 响应式设计,手机友好

## 技术栈

- **GitHub Actions**: 定时任务和自动化
- **GitHub Pages**: 静态网站托管
- **Python**: 数据处理和页面生成
- **Tailwind CSS**: 页面样式

## 访问地址

- **网站**: https://你的用户名.github.io/ai-daily-report/
- **GitHub仓库**: [查看代码](https://github.com/你的用户名/ai-daily-report)

## 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 生成今日日报
python generate.py

# 查看生成的网站
open index.html
```
